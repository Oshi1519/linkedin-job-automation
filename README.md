# linkedin-job-automation
Automated LinkedIn job search using Python &amp; Selenium
## Features
- Automated LinkedIn login
- Searches Web Developer job posts from past 24 hours
- Extracts recruiter name, title, contact details
- Saves results to CSV automatically

## Tech Stack
- Python
- Selenium
- ChromeDriver
- WebDriver Manager

## How to Run
pip install selenium webdriver-manager
python linkedin_search.py

## Output
Extracts recruiter data including:
- Name
- Title
- Email
- Phone
- Post Content
